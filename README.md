# WORDS SEARCH

Aradığınız kelimeyi aratarak anlamını hızlıca öğrenebilirsiniz.Örnekleri ile pekiştirme yapabilirsiniz. Bir kelimenin eş anlamlısını bilmek ister misiniz? WORDS SEARCH ile öğrenebilirsiniz. Aradığınız kelimenin eş anlamlılarını direk görebilirsiniz. Ayrıca kelimelerin telaffuzlarını dinleyebilirsiniz.

## Özellikler

 ✔️ Açılış ekranında internetinizi tekrar tekrar kontrol edebilir ve internetiniz geldiğinde devam edebilirsiniz.

✔️ Aranan son 5 kelime Arama Ekranına gelir ve aramanız listeye eklenir.

✔️ Arama işleminde aranan kelime varsa Detay Ekranına gider yoksa. Bir uyarı mesajı verir.

✔️ Detay sayfasında açıklamalar, telaffuz ve eş anlamlı kelimeler bulunur.

✔️ Açıklamalar filtrelenebilir

## Uygulamanın İçeriği

✔️ MVMM Tasarım Mimarisi

✔️ Core Data

✔️ AVFoundation

✔️Modül: DictionaryAPI
